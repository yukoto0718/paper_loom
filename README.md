# paper_loom

#### work-in-progress

Paper-Loom - 学術論文向けPDFリーダーの試作（開発初期段階） 、3つのAI機能を構想中：
```
>OCRによるPDF最適化・読みやすい形式への変換
>プロンプトベースのAI質問機能
>引用関係による論文推薦システム
```
技術構成：Flutter（フロントエンド）+ Python FastAPI（バックエンド）
